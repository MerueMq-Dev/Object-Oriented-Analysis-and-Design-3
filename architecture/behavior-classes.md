# Классы поведения

## GameController 
Назначение: управление процессом игры.

#### Запросы
- GetCurrentState() — текущее состояние игры

#### Команды
- ProcessMove(move) — обработать ход
- StartGame() — начать игру
- EndGame() — завершить игру

#### Ограничения
- ход возможен только в состоянии "игра"


## GameStateMachine
Назначение: управление режимами игры.

#### Запросы
- GetState() — текущий режим

#### Команды
- ChangeState(newState) — сменить режим

#### Ограничения
- переход возможен только между допустимыми состояниями


## GameBoard
Назначение: хранение состояния игрового поля.

#### Запросы
- GetElement(x, y) — вернуть элемент в клетке
- IsInside(x, y) — находится ли клетка в пределах поля
- HasMatch() — есть ли совпадения на поле

#### Команды
- Swap(element1, element2) — поменять элементы местами
- RemoveMatches() — удалить найденные совпадения
- FillEmpty() — заполнить пустые клетки новыми элементами

#### Ограничения
- размер поля фиксирован (8×8)
- в каждой клетке не более одного элемента



## GameSession
Назначение: данные и состояние одной партии.

#### Запросы
- GetScore() — текущие очки
- IsFinished() — завершена ли партия

#### Команды
- Start() — начать партию
- Finish() — завершить партию
- AddScore(value) — увеличить счёт

#### Ограничения
- счёт не может быть отрицательным
- партия имеет состояние: начата или завершена

### GameElement
Назначение: элемент игрового поля.

#### Запросы
- GetType() — тип/цвет элемента
Команды
- SetType(type) — изменить тип элемента

#### Ограничения
элемент всегда имеет допустимый тип


### Leaderboard
Назначение: хранение результатов игроков.

#### Запросы
- GetTop(n) — получить лучшие результаты
- GetAll() — список всех результатов

#### Команды
- AddResult(player, score) — добавить результат

#### Ограничения
- результаты упорядочены по убыванию очков
