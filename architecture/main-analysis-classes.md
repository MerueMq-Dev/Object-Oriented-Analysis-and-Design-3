# Основные классы анализа

Game — игра «Три в ряд», её правила и игровой процесс

GameSession — одна игровая партия с текущим состоянием поля и счётом

Player — игрок, который играет в партии

GameBoard — игровое поле, на котором размещены элементы

GameElement — отдельная фишка на поле (кристалл, конфета и т.п.)

Match — совпадение трёх и более элементов одного типа

Score — счёт игрока, набранный в ходе партии
