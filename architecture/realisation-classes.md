# Классы реализации

Для игры «Три в ряд» потребуется несколько специфических классов реализации, 
хотя большую часть можно реализовать с помощью стандартных структур данных .NET.

## GameBoard
Двумерная структура для хранения игрового поля. Под капотом может использовать двумерный массив или список списков для быстрого доступа к элементам и соседним клеткам.

## MatchGroup
Группа совпавших элементов. Внутри себя может использовать список координат или словарь для эффективной проверки принадлежности элемента к совпадению.
